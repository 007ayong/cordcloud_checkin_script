# CordCloud 自动签到脚本

## 📋 简介

这是一个用于 CordCloud (https://cordcloud.biz) 的自动签到脚本，专为 **ScriptCat 后台定时执行** 设计。

## ✨ 功能特性

- ✅ 自动签到 - 每日自动执行签到任务
- 🔔 浏览器通知 - 签到结果实时推送
- ⚡ 后台运行 - 支持ScriptCat后台定时执行
- 🔄 智能重试 - 网络失败自动重试（最多3次）
- 🛡️ 登录检测 - 自动检测登录状态

## 🚀 安装方法

### 1. 安装脚本管理器

首先需要安装 **ScriptCat (脚本猫)** 浏览器扩展：
- **ScriptCat (推荐)**: [官网下载](https://scriptcat.org/)
- **WinApps 下载 ScriptCat**：https://www.winapps.cc/scriptcat.html

### 2. 安装签到脚本

1. 打开 ScriptCat 扩展
2. 点击"添加新脚本"
3. 将 `cordcloud-checkin.user.js` 内容复制到编辑器中
4. 保存脚本

### 3. 配置定时任务

1. 在脚本列表中找到脚本
2. 点击"设置"→"定时任务"
3. 添加 cron 表达式，如 `0 8 * * *`（每天早上8点）
4. 保存配置

### 4. 授权通知权限

首次运行时，浏览器会请求通知权限，请点击"允许"以接收签到结果通知。

## 📖 使用指南

### 安装和配置

1. **安装脚本管理器**
   - 下载并安装 [ScriptCat (脚本猫)](https://scriptcat.org/)

2. **安装签到脚本**
   - 打开 ScriptCat 扩展
   - 点击"添加新脚本"
   - 将 `cordcloud-checkin.user.js` 内容复制到编辑器中
   - 保存脚本

3. **配置定时任务**
   - 在脚本列表中找到脚本
   - 点击"设置"→"定时任务"
   - 添加 cron 表达式，如 `0 8 * * *`（每天早上8点）
   - 保存配置

4. **授权通知权限**
   - 首次运行时，浏览器会请求通知权限
   - 点击"允许"以接收签到结果通知

### 手动触发签到

1. 点击 ScriptCat 图标
2. 找到脚本的菜单命令
3. 点击"立即签到"

## ⚙️ 配置说明

脚本默认配置：
- **签到时间**: 每天 11:00-23:00 之间随机执行一次（由 cron 表达式控制）
- **通知**: 签到结果通过浏览器通知发送
- **超时**: 15秒
- **重试**: 最多3次重试

## 🐛 常见问题

### 定时签到不工作？

1. 检查 ScriptCat 后台运行权限
2. 验证定时任务 cron 表达式
3. 检查网络连接
4. 查看浏览器控制台日志

### 签到失败？

1. 检查是否已登录 CordCloud 账户
2. 确保浏览器通知权限已开启
3. 打开浏览器控制台 (F12) 查看错误信息
4. 检查网络连接是否正常
